=====================================
  如何更新网站内容
=====================================

方法1：使用GitHub Desktop（推荐）
---------------------------------
1. 修改文件（index.html、style.css等）
2. 打开GitHub Desktop
3. 左下角输入更新说明（Summary）
4. 点击 "Commit to main"
5. 点击右上角 "Push origin"
6. 等待1-2分钟，网站自动更新

方法2：使用命令行
---------------------------------
在项目文件夹中打开PowerShell：

git add .
git commit -m "更新简历内容"
git push

方法3：使用部署脚本
---------------------------------
双击运行 deploy.ps1 脚本，选择"更新部署"

=====================================
  常见问题
=====================================

Q: 修改后网站没有更新？
A: 1. 清除浏览器缓存（Ctrl+F5强制刷新）
   2. 等待2-3分钟（GitHub Pages需要重新构建）
   3. 检查GitHub仓库是否已更新

Q: 如何修改网站内容？
A: 1. 个人信息：编辑 index.html
   2. 样式颜色：编辑 style.css 的 :root 部分
   3. 交互功能：编辑 script.js

Q: 如何更换头像？
A: 替换 avatar.png 文件，推送到GitHub即可

Q: 如何添加新的内容板块？
A: 1. 在 index.html 中添加新的 <section>
   2. 在导航栏添加对应链接
   3. 根据需要添加CSS样式
