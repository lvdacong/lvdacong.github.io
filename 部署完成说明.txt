=====================================
  GitHub Pages 配置步骤
=====================================

推送代码成功后，请完成以下配置：

第1步：访问GitHub Pages设置
---------------------------------
已自动为您打开页面，或手动访问：
https://github.com/lvdacong/lvdacong.github.io/settings/pages

第2步：配置部署源
---------------------------------
在 "Build and deployment" 部分：

Source (源): 
  选择 "Deploy from a branch"

Branch (分支):
  Branch: 选择 "main"
  Folder: 选择 "/ (root)"
  
然后点击 "Save" 按钮

第3步：等待部署完成
---------------------------------
保存后，页面会显示：
"Your site is ready to be published at https://lvdacong.github.io/"

等待 1-2 分钟后，刷新页面会变成：
"Your site is live at https://lvdacong.github.io/"

第4步：访问您的网站
---------------------------------
在浏览器中打开：
https://lvdacong.github.io

恭喜！您的个人简历网页已经成功部署！🎉

=====================================
  后续更新网站的方法
=====================================

当您修改简历内容后，只需在项目文件夹中执行：

1. 打开PowerShell
2. 运行以下命令：

   git add .
   git commit -m "更新简历内容"
   git push

3. 等待1-2分钟后网站自动更新

或者直接双击运行 deploy.ps1 脚本！
